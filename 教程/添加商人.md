# 添加商人

你可以添加自定义商人到游戏中，让玩家可以购买特殊物品。

**源文件：** 
- [13AddTraderWithAssortJson/](../13AddTraderWithAssortJson/) - JSON 方式
- [13.1AddTraderWithDynamicAssorts/](../13.1AddTraderWithDynamicAssorts/) - 动态方式

## 商人数据结构

一个商人需要以下数据：

| 数据 | 说明 |
|------|------|
| 基础信息 | 名称、头像、解锁等级等 |
| 物品列表 (Assort) | 出售的物品 |
| 价格方案 | 物品的价格（卢布或物物交换） |
| 忠诚等级 | 购买物品所需的忠诚等级 |
| 本地化文本 | 各语言的商人名称和描述 |

### 2. 加载商人代码

```csharp
using System.Reflection;
using SPTarkov.DI.Annotations;
using SPTarkov.Server.Core.DI;
using SPTarkov.Server.Core.Helpers;
using SPTarkov.Server.Core.Models.Eft.Common.Tables;
using SPTarkov.Server.Core.Models.Spt.Mod;
using SPTarkov.Server.Core.Models.Utils;
using SPTarkov.Server.Core.Routers;
using SPTarkov.Server.Core.Servers;
using SPTarkov.Server.Core.Services;
using SPTarkov.Server.Core.Utils;
using Path = System.IO.Path;

[Injectable(TypePriority = OnLoadOrder.PostDBModLoader + 1)]
public class AddTrader(
    ISptLogger<AddTrader> logger,
    ModHelper modHelper,
    ImageRouter imageRouter,
    ConfigServer configServer,
    TimeUtil timeUtil,
    AddCustomTraderHelper traderHelper
) : IOnLoad
{
    private readonly TraderConfig _traderConfig = configServer.GetConfig<TraderConfig>();
    private readonly RagfairConfig _ragfairConfig = configServer.GetConfig<RagfairConfig>();

    public Task OnLoad()
    {
        // 获取模组路径
        var pathToMod = modHelper.GetAbsolutePathToModFolder(Assembly.GetExecutingAssembly());
        
        // 加载商人基础配置
        var traderBase = modHelper.GetJsonDataFromFile<TraderBase>(pathToMod, "data/base.json");
        
        // 注册头像
        var imagePath = Path.Combine(pathToMod, "data/cat.jpg");
        imageRouter.AddRoute(traderBase.Avatar.Replace(".jpg", ""), imagePath);
        
        // 设置刷新时间
        traderHelper.SetTraderUpdateTime(_traderConfig, traderBase, 
            timeUtil.GetHoursAsSeconds(1), timeUtil.GetHoursAsSeconds(2));
        
        // 添加到跳蚤市场
        _ragfairConfig.Traders.TryAdd(traderBase.Id, true);
        
        // 添加商人到数据库
        traderHelper.AddTraderWithEmptyAssortToDb(traderBase);
        
        // 添加本地化
        traderHelper.AddTraderToLocales(traderBase, "Cat", "这是猫咪商店");
        
        // 加载物品配置
        var assort = modHelper.GetJsonDataFromFile<TraderAssort>(pathToMod, "data/assort.json");
        traderHelper.OverwriteTraderAssort(traderBase.Id, assort);
        
        logger.Success("商人添加成功！");
        return Task.CompletedTask;
    }
}
```

## 方式二：动态创建

适合需要根据条件动态生成物品的场景。

### FluentTraderAssortCreator 流式 API

```csharp
using SPTarkov.Server.Core.Models.Enums;

// 添加简单物品：牛奶，2000卢布，忠诚1级
fluentAssortCreator
    .CreateSingleAssortItem(ItemTpl.DRINK_PACK_OF_MILK)
    .AddStackCount(200)           // 库存数量
    .AddBuyRestriction(10)        // 限购10个
    .AddMoneyCost(Money.ROUBLES, 2000)  // 价格
    .AddLoyaltyLevel(1)           // 忠诚等级要求
    .Export(traderBase.Id);

// 添加物物交换：3比特币 + 1医疗包换牛奶
fluentAssortCreator
    .CreateSingleAssortItem(ItemTpl.DRINK_PACK_OF_MILK)
    .AddStackCount(100)
    .AddBarterCost(ItemTpl.BARTER_PHYSICAL_BITCOIN, 3)
    .AddBarterCost(ItemTpl.MEDKIT_SALEWA_FIRST_AID_KIT, 1)
    .AddLoyaltyLevel(1)
    .Export(traderBase.Id);
```

## 小贴士

1. **商人ID要唯一**：避免和游戏内商人冲突
2. **必须添加到跳蚤**：`_ragfairConfig.Traders.TryAdd(traderId, true)`
3. **本地化用 AddTransformer**：因为本地化数据是延迟加载的
4. **克隆预设**：使用预设武器时必须用 `ICloner` 深拷贝
5. **ItemTpl 常量**：用常量代替字符串ID，代码更清晰
