# 生命周期

SPT 模组可以在不同的时机执行代码。了解生命周期可以让你在正确的时机做正确的事情。

## 两种生命周期接口

### IOnLoad - 加载时执行

模组加载时执行一次，适合做初始化工作。

**源文件：** [8OnLoad/OnLoadExample.cs](../8OnLoad/OnLoadExample.cs)

```csharp
using SPTarkov.DI.Annotations;
using SPTarkov.Server.Core.DI;
using SPTarkov.Server.Core.Models.Utils;

[Injectable(TypePriority = OnLoadOrder.PostDBModLoader)]
public class MyMod(
    ISptLogger<MyMod> logger
) : IOnLoad  // 实现 IOnLoad 接口
{
    public Task OnLoad()
    {
        // 这里的代码在加载时执行一次
        logger.Success("模组已加载！");
        return Task.CompletedTask;
    }
}
```

### IOnUpdate - 循环执行

服务器运行期间反复执行，适合做定时任务。

**源文件：** [9OnUpdate/OnUpdateExample.cs](../9OnUpdate/OnUpdateExample.cs)

```csharp
using SPTarkov.DI.Annotations;
using SPTarkov.Server.Core.DI;
using SPTarkov.Server.Core.Models.Utils;

[Injectable(TypePriority = OnUpdateOrder.InsuranceCallbacks)]
public class MyUpdateMod(
    ISptLogger<MyUpdateMod> logger
) : IOnUpdate  // 实现 IOnUpdate 接口
{
    public Task<bool> OnUpdate(long timeSinceLastRun)
    {
        // timeSinceLastRun 是距离上次执行的时间（毫秒）
        
        // 返回 true 表示成功，false 表示失败
        return Task.FromResult(true);
    }
}
```

## 加载顺序（OnLoadOrder）

加载顺序决定你的代码什么时候执行：

| 优先级常量 | 说明 |
|-----------|------|
| `PreSptModLoader` | SPT 模组加载器之前 |
| `PostSptModLoader` | SPT 模组加载器之后 |
| `PostDBModLoader` | 数据库加载之后 |
| `Watermark` | 水印阶段 |

### 使用方式

```csharp
// 数据库加载后立即执行（数据是原始状态）
[Injectable(TypePriority = OnLoadOrder.PostDBModLoader)]

// 数据库加载后 +1（在 PostDBModLoader 之后执行）
[Injectable(TypePriority = OnLoadOrder.PostDBModLoader + 1)]

// SPT 处理完成后执行
[Injectable(TypePriority = OnLoadOrder.PostSptModLoader + 1)]
```

### 时机选择

```csharp
// 需要修改原始数据库数据
[Injectable(TypePriority = OnLoadOrder.PostDBModLoader + 1)]
public class EarlyMod : IOnLoad { }

// 需要 SPT 处理后的数据
[Injectable(TypePriority = OnLoadOrder.PostSptModLoader + 1)]
public class LateMod : IOnLoad { }

// 需要在其他模组之前执行
[Injectable(TypePriority = OnLoadOrder.PreSptModLoader + 1)]
public class FirstMod : IOnLoad { }
```

## 更新顺序（OnUpdateOrder）

更新顺序决定你的更新逻辑在哪个阶段执行：

```csharp
// 在保险回调之后执行
[Injectable(TypePriority = OnUpdateOrder.InsuranceCallbacks)]

// 在保存回调之后执行
[Injectable(TypePriority = OnUpdateOrder.SaveCallbacks)]
```

### 控制执行频率

```csharp
public class TimedUpdate : IOnUpdate
{
    private long _lastRunTime = 0;
    private const long Interval = 60000; // 60秒

    public Task<bool> OnUpdate(long timeSinceLastRun)
    {
        // 累计时间
        _lastRunTime += timeSinceLastRun;
        
        // 每60秒执行一次
        if (_lastRunTime >= Interval)
        {
            _lastRunTime = 0;
            // 执行定时任务...
        }
        
        return Task.FromResult(true);
    }
}
```

## 完整示例

### 示例1：初始化任务

```csharp
using SPTarkov.DI.Annotations;
using SPTarkov.Server.Core.DI;
using SPTarkov.Server.Core.Models.Spt.Mod;
using SPTarkov.Server.Core.Models.Utils;
using SPTarkov.Server.Core.Services;

namespace MyMod;

public record ModMetadata : AbstractModMetadata
{
    public override string ModGuid { get; init; } = "com.example.lifecycle";
    public override string Name { get; init; } = "生命周期示例";
    public override string Author { get; init; } = "开发者";
    public override SemanticVersioning.Version Version { get; init; } = new("1.0.0");
    public override SemanticVersioning.Range SptVersion { get; init; } = new("~4.0.0");
    public override string License { get; init; } = "MIT";
}

[Injectable(TypePriority = OnLoadOrder.PostDBModLoader)]
public class InitMod(
    ISptLogger<InitMod> logger,
    DatabaseService databaseService
) : IOnLoad
{
    public Task OnLoad()
    {
        logger.Success("模组初始化开始...");
        
        // 读取数据库
        var globals = databaseService.GetGlobals();
        
        // 修改配置
        globals.Configuration.SavagePlayCooldown = 1;
        
        logger.Success("模组初始化完成！");
        return Task.CompletedTask;
    }
}
```

### 示例2：定时任务

```csharp
[Injectable(TypePriority = OnUpdateOrder.InsuranceCallbacks)]
public class TimedTaskMod(
    ISptLogger<TimedTaskMod> logger
) : IOnUpdate
{
    private long _accumulator = 0;
    private const long CheckInterval = 60000; // 每分钟检查一次

    public Task<bool> OnUpdate(long timeSinceLastRun)
    {
        _accumulator += timeSinceLastRun;
        
        if (_accumulator >= CheckInterval)
        {
            _accumulator = 0;
            
            // 执行定时检查
            logger.Info("执行定时检查...");
            
            // 返回 false 表示失败，会记录错误
            return Task.FromResult(true);
        }
        
        return Task.FromResult(true);
    }
}
```

## 小贴士

1. **选择正确时机**：数据操作要在 `PostDBModLoader` 之后
2. **避免阻塞**：OnUpdate 不要执行耗时操作，会影响服务器性能
3. **控制频率**：使用累加器控制更新频率
4. **返回值**：OnUpdate 返回 `true` 表示成功
5. **异步支持**：两个接口都支持异步操作
