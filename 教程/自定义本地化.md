# 自定义本地化

你可以添加或修改游戏的本地化文本，支持多种语言。

**源文件：** [25AddCustomLocales/AddCustomLocales.cs](../25AddCustomLocales/AddCustomLocales.cs)

## 应用场景

- 为自定义物品添加多语言名称
- 修改游戏内文本
- 为模组添加本地化支持

## 本地化数据特点

本地化数据是**延迟加载**的，必须使用 `AddTransformer` 方法来添加或修改文本。

## 基本用法

### 添加本地化文本

```csharp
using SPTarkov.DI.Annotations;
using SPTarkov.Server.Core.DI;
using SPTarkov.Server.Core.Models.Spt.Mod;
using SPTarkov.Server.Core.Models.Utils;
using SPTarkov.Server.Core.Services;

[Injectable(TypePriority = OnLoadOrder.PostDBModLoader + 1)]
public class LocaleMod(
    ISptLogger<LocaleMod> logger,
    DatabaseService databaseService,
    LocaleService localeService,
    ServerLocalisationService serverLocalisationService
) : IOnLoad
{
    public Task OnLoad()
    {
        // 添加/修改英文文本
        if (databaseService.GetLocales().Global.TryGetValue("en", out var englishLocale))
        {
            englishLocale.AddTransformer(localeData =>
            {
                // 添加新文本
                localeData.Add("MyMod_Welcome", "Welcome to my server!");
                localeData.Add("MyMod_Item_Name", "Custom Item");
                localeData.Add("MyMod_Item_Desc", "This is a custom item from my mod.");
                
                // 修改现有文本
                localeData["Attention! This is a Beta version..."] = 
                    "欢迎来到我的服务器！";
                
                return localeData;
            });
        }
        
        // 添加/修改中文文本
        if (databaseService.GetLocales().Global.TryGetValue("zh", out var chineseLocale))
        {
            chineseLocale.AddTransformer(localeData =>
            {
                localeData.Add("MyMod_Welcome", "欢迎来到我的服务器！");
                localeData.Add("MyMod_Item_Name", "自定义物品");
                localeData.Add("MyMod_Item_Desc", "这是一个来自我的模组的自定义物品。");
                
                return localeData;
            });
        }
        
        logger.Success("本地化文本添加完成！");
        return Task.CompletedTask;
    }
}
```

### 读取本地化文本

```csharp
// 方式1：使用 LocaleService（需要指定语言）
var localeDb = localeService.GetLocaleDb("en");
var welcomeText = localeDb["MyMod_Welcome"];

// 方式2：使用 ServerLocalisationService（自动适配玩家语言）
var localizedText = serverLocalisationService.GetText("MyMod_Welcome");
```

## 完整示例

```csharp
using SPTarkov.DI.Annotations;
using SPTarkov.Server.Core.DI;
using SPTarkov.Server.Core.Models.Spt.Mod;
using SPTarkov.Server.Core.Models.Utils;
using SPTarkov.Server.Core.Services;

namespace MyMod;

public record ModMetadata : AbstractModMetadata
{
    public override string ModGuid { get; init; } = "com.example.locales";
    public override string Name { get; init; } = "本地化示例";
    public override string Author { get; init; } = "开发者";
    public override SemanticVersioning.Version Version { get; init; } = new("1.0.0");
    public override SemanticVersioning.Range SptVersion { get; init; } = new("~4.0.0");
    public override string License { get; init; } = "MIT";
}

[Injectable(TypePriority = OnLoadOrder.PostDBModLoader + 1)]
public class LocaleExample(
    ISptLogger<LocaleExample> logger,
    DatabaseService databaseService,
    LocaleService localeService,
    ServerLocalisationService serverLocalisationService
) : IOnLoad
{
    public Task OnLoad()
    {
        // 添加多种语言的文本
        AddLocales();
        
        // 验证添加结果
        VerifyLocales();
        
        logger.Success("本地化设置完成！");
        return Task.CompletedTask;
    }
    
    private void AddLocales()
    {
        var localeTexts = new Dictionary<string, Dictionary<string, string>>
        {
            // 英文
            ["en"] = new()
            {
                ["MyMod_Title"] = "My Awesome Mod",
                ["MyMod_Description"] = "This mod adds awesome features!",
                ["MyMod_Command_Help"] = "Type 'help' for available commands"
            },
            // 中文
            ["zh"] = new()
            {
                ["MyMod_Title"] = "我的超棒模组",
                ["MyMod_Description"] = "这个模组添加了超棒的功能！",
                ["MyMod_Command_Help"] = "输入 'help' 查看可用命令"
            }
        };
        
        foreach (var (lang, texts) in localeTexts)
        {
            if (databaseService.GetLocales().Global.TryGetValue(lang, out var locale))
            {
                locale.AddTransformer(localeData =>
                {
                    foreach (var (key, value) in texts)
                    {
                        localeData.Add(key, value);
                    }
                    return localeData;
                });
            }
        }
    }
    
    private void VerifyLocales()
    {
        // 获取并显示本地化文本
        var enLocale = localeService.GetLocaleDb("en");
        var zhLocale = localeService.GetLocaleDb("zh");
        
        logger.Info($"英文标题: {enLocale["MyMod_Title"]}");
        logger.Info($"中文标题: {zhLocale["MyMod_Title"]}");
        
        // 使用服务器本地化服务（自动适配）
        var autoText = serverLocalisationService.GetText("MyMod_Title");
        logger.Info($"自动适配: {autoText}");
    }
}
```

## 商人本地化示例

为自定义商人添加本地化：

```csharp
// 在添加商人时设置本地化
traderHelper.AddTraderToLocales(traderBase, "Cat", "这是猫咪商店");

// 或者手动添加
databaseService.GetLocales().Global["en"].AddTransformer(localeData =>
{
    localeData.Add($"{traderId} FullName", "Cat Trader");
    localeData.Add($"{traderId} FirstName", "Cat");
    localeData.Add($"{traderId} Nickname", "Kitty");
    localeData.Add($"{traderId} Location", "In the cat shop");
    localeData.Add($"{traderId} Description", "Welcome to the cat shop. Meow!");
    
    return localeData;
});
```

## 小贴士

1. **必须用 AddTransformer**：本地化数据延迟加载，直接修改无效
2. **键名要有前缀**：使用模组名作为前缀避免冲突，如 `MyMod_ItemName`
3. **支持多语言**：为常用语言提供翻译，让更多玩家使用
4. **验证添加结果**：添加后读取验证是否成功
5. **使用 ServerLocalisationService**：自动适配玩家语言设置
