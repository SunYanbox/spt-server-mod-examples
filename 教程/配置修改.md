# 配置修改

SPT 服务器有很多配置文件控制游戏行为，比如机器人数量、天气、空投等。你可以通过修改这些配置来自定义游戏体验。

**源文件：** [3EditSptConfig/EditConfigs.cs](../3EditSptConfig/EditConfigs.cs)

## 配置类型

常用的配置类型：

| 配置类 | 说明 |
|--------|------|
| `BotConfig` | 机器人相关设置（数量上限、战利品等） |
| `HideoutConfig` | 藏身处设置（建造时间、制作时间） |
| `WeatherConfig` | 天气设置（季节、天气类型） |
| `AirdropConfig` | 空投设置（类型概率、内容） |
| `PmcChatResponse` | PMC 发送消息的概率 |
| `QuestConfig` | 任务相关设置 |
| `PmcConfig` | PMC 行为和战利品设置 |
| `TraderConfig` | 商人相关设置 |

## 获取配置

通过 `ConfigServer` 获取配置(预计SPT4.1.x版本无法这样做)：

```csharp
using SPTarkov.Server.Core.Models.Spt.Config;
using SPTarkov.Server.Core.Servers;

public class MyMod(
    ConfigServer configServer
) : IOnLoad
{
    // 获取各类配置
    private readonly BotConfig _botConfig = configServer.GetConfig<BotConfig>();
    private readonly WeatherConfig _weatherConfig = configServer.GetConfig<WeatherConfig>();
    private readonly HideoutConfig _hideoutConfig = configServer.GetConfig<HideoutConfig>();
    
    public Task OnLoad()
    {
        // 使用配置...
        return Task.CompletedTask;
    }
}
```

## 常用配置修改示例

### 1. 天气配置

强制设置为冬天：

```csharp
using SPTarkov.Server.Core.Models.Enums;

_weatherConfig.OverrideSeason = Season.WINTER;
```

可选季节：`SPRING`、`SUMMER`、`AUTUMN`、`WINTER`

### 2. 藏身处配置

让所有制作和建造都很快：

```csharp
// 所有制作 60 秒完成
_hideoutConfig.OverrideCraftTimeSeconds = 60;

// 所有建造 60 秒完成
_hideoutConfig.OverrideBuildTimeSeconds = 60;
```

### 3. 空投配置

让武器装甲空投超级常见：

```csharp
using SPTarkov.Server.Core.Extensions;
using SPTarkov.Server.Core.Models.Enums;

// 大幅提高武器装甲空投权重
_airdropConfig.AirdropTypeWeightings.AddOrUpdate(SptAirdropTypeEnum.weaponArmor, 999);

// 设置空投内容
if (_airdropConfig.Loot.TryGetValue("weaponArmor", out var lootSettings))
{
    lootSettings.WeaponCrateCount.Min = 3;  // 最少 3 个武器箱
    lootSettings.WeaponCrateCount.Max = 3;  // 最多 3 个武器箱
}
```

### 4. 机器人配置

调整机器人数上限和战利品：

```csharp
using SPTarkov.Server.Core.Extensions;

// 增加立交桥的机器人上限
_botConfig.MaxBotCap.AddOrUpdate("interchange", 50);

// 禁止拾荒者携带战利品
_botConfig.DisableLootOnBotTypes.Add("assault");
```

### 5. PMC 配置

让 PMC 口袋里装满贵重物品：

```csharp
_pmcConfig.LootSettings.Pocket.TotalRubByLevel =
[
    new MinMaxLootValue
    {
        Min = 1,
        Max = 99,
        Value = 9999999  // 口袋物品价值上限
    }
];
```

### 6. PMC 消息回复

让 PMC 击杀你后 100% 发消息：

```csharp
_pmcChatResponseConfig.Killer.ResponseChancePercent = 100;
```

### 7. 任务配置

延长任务奖励邮件的有效期：

```csharp
using SPTarkov.Server.Core.Extensions;

// 未听闻版玩家邮件有效期 168 小时（7天）
_questConfig.MailRedeemTimeHours.AddOrUpdate("unheard_edition", 168);
```

## 完整示例

```csharp
using SPTarkov.DI.Annotations;
using SPTarkov.Server.Core.DI;
using SPTarkov.Server.Core.Extensions;
using SPTarkov.Server.Core.Models.Enums;
using SPTarkov.Server.Core.Models.Spt.Config;
using SPTarkov.Server.Core.Models.Spt.Mod;
using SPTarkov.Server.Core.Models.Utils;
using SPTarkov.Server.Core.Servers;

namespace MyMod;

public record ModMetadata : AbstractModMetadata
{
    public override string ModGuid { get; init; } = "com.example.editconfig";
    public override string Name { get; init; } = "配置修改示例";
    public override string Author { get; init; } = "开发者";
    public override SemanticVersioning.Version Version { get; init; } = new("1.0.0");
    public override SemanticVersioning.Range SptVersion { get; init; } = new("~4.0.0");
    public override string License { get; init; } = "MIT";
}

[Injectable(TypePriority = OnLoadOrder.PostDBModLoader + 1)]
public class EditConfig(
    ConfigServer configServer,
    ISptLogger<EditConfig> logger
) : IOnLoad
{
    private readonly BotConfig _botConfig = configServer.GetConfig<BotConfig>();
    private readonly HideoutConfig _hideoutConfig = configServer.GetConfig<HideoutConfig>();
    private readonly WeatherConfig _weatherConfig = configServer.GetConfig<WeatherConfig>();

    public Task OnLoad()
    {
        // 强制冬天
        _weatherConfig.OverrideSeason = Season.WINTER;
        
        // 快速建造和制作
        _hideoutConfig.OverrideCraftTimeSeconds = 60;
        _hideoutConfig.OverrideBuildTimeSeconds = 60;
        
        // 增加机器人数
        _botConfig.MaxBotCap.AddOrUpdate("interchange", 50);
        
        logger.Success("配置修改完成！");
        return Task.CompletedTask;
    }
}
```

## AddOrUpdate 扩展方法

修改字典时常使用 `AddOrUpdate` 方法：

```csharp
// 如果键不存在，添加；如果存在，更新
dictionary.AddOrUpdate("key", newValue);
```

这比传统的先检查再添加/更新更简洁：

```csharp
// 传统写法（不推荐）
if (dictionary.ContainsKey("key"))
    dictionary["key"] = newValue;
else
    dictionary.Add("key", newValue);

// 使用 AddOrUpdate（推荐）
dictionary.AddOrUpdate("key", newValue);
```

## 小贴士

1. **配置名要准确**：`GetConfig<BotConfig>()` 中的类型要正确
2. **使用 readonly**：配置字段建议用 `readonly`，因为你是在修改配置内容，不是替换整个配置
3. **时机很重要**：配置修改要在数据库加载后进行，使用 `OnLoadOrder.PostDBModLoader + 1`
4. **测试验证**：修改后进入游戏验证效果
