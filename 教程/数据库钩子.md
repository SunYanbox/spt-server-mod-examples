# 数据库钩子

有时你需要在数据库加载的不同阶段执行操作。SPT 提供了多个加载时机，让你可以在正确的时机修改数据。

**源文件：** [14AfterDBLoadHook/AfterDBLoadHook.cs](../14AfterDBLoadHook/AfterDBLoadHook.cs)

## 加载时机

| 时机 | 常量 | 数据状态 |
|------|------|---------|
| 数据库刚加载 | `OnLoadOrder.PostDBModLoader` | 原始 JSON 数据 |
| SPT 处理完成 | `OnLoadOrder.PostSptModLoader` | 已被 SPT 逻辑处理 |

## PostDBModLoader - 原始数据阶段

数据库从 JSON 文件加载完成，但 SPT 的核心逻辑还没有处理数据。

```csharp
using SPTarkov.DI.Annotations;
using SPTarkov.Server.Core.DI;
using SPTarkov.Server.Core.Models.Common;
using SPTarkov.Server.Core.Models.Eft.Common.Tables;
using SPTarkov.Server.Core.Models.Spt.Mod;
using SPTarkov.Server.Core.Models.Utils;
using SPTarkov.Server.Core.Servers;

[Injectable(TypePriority = OnLoadOrder.PostDBModLoader + 1)]
public class EarlyDbHook(
    DatabaseServer databaseServer,
    ISptLogger<EearlyDbHook> logger
) : IOnLoad
{
    public Task OnLoad()
    {
        var items = databaseServer.GetTables().Templates.Items;
        
        // 数据是原始状态，SPT 逻辑尚未处理
        logger.Info($"数据库物品数量: {items.Count}");
        
        // 在 SPT 处理之前修改数据
        if (items.TryGetValue(ItemTpl.NIGHTVISION_L3HARRIS_GPNVG18_NIGHT_VISION_GOGGLES, out var nvgs))
        {
            nvgs.Properties.CanSellOnRagfair = true;
        }
        
        return Task.CompletedTask;
    }
}
```

## PostSptModLoader - 处理后阶段

SPT 的核心逻辑已经处理完数据，所有初始化都完成了。

```csharp
[Injectable(TypePriority = OnLoadOrder.PostSptModLoader + 1)]
public class LateDbHook(
    DatabaseServer databaseServer,
    ISptLogger<LateDbHook> logger
) : IOnLoad
{
    public Task OnLoad()
    {
        var items = databaseServer.GetTables().Templates.Items;
        
        // 此时之前的修改已被 SPT 处理
        // 可以验证修改效果或进行后续操作
        
        if (items.TryGetValue(ItemTpl.NIGHTVISION_L3HARRIS_GPNVG18_NIGHT_VISION_GOGGLES, out var nvgs))
        {
            logger.Info($"NVG 跳蚤可售: {nvgs.Properties.CanSellOnRagfair}");
        }
        
        return Task.CompletedTask;
    }
}
```

## 使用场景对比

### 使用 PostDBModLoader

- 修改原始数据库数据
- 在 SPT 处理前调整数据
- 添加新物品/商人

### 使用 PostSptModLoader

- 验证修改结果
- 基于已处理数据做进一步操作
- 与其他模组兼容（确保执行顺序）

## 完整示例：监控数据变化

```csharp
using SPTarkov.DI.Annotations;
using SPTarkov.Server.Core.DI;
using SPTarkov.Server.Core.Models.Common;
using SPTarkov.Server.Core.Models.Eft.Common.Tables;
using SPTarkov.Server.Core.Models.Logging;
using SPTarkov.Server.Core.Models.Spt.Mod;
using SPTarkov.Server.Core.Models.Utils;
using SPTarkov.Server.Core.Servers;

namespace MyMod;

public record ModMetadata : AbstractModMetadata
{
    public override string ModGuid { get; init; } = "com.example.dbhook";
    public override string Name { get; init; } = "数据库钩子示例";
    public override string Author { get; init; } = "开发者";
    public override SemanticVersioning.Version Version { get; init; } = new("1.0.0");
    public override SemanticVersioning.Range SptVersion { get; init; } = new("~4.0.0");
    public override string License { get; init; } = "MIT";
}

// 早期钩子：数据库加载后立即执行
[Injectable(TypePriority = OnLoadOrder.PostDBModLoader + 1)]
public class EarlyHook(
    DatabaseServer databaseServer,
    ISptLogger<EarlyHook> logger
) : IOnLoad
{
    private Dictionary<MongoId, TemplateItem>? _itemsDb;

    public Task OnLoad()
    {
        _itemsDb = databaseServer.GetTables().Templates.Items;
        
        logger.LogWithColor($"数据库物品数量: {_itemsDb.Count}", 
            LogTextColor.Red, LogBackgroundColor.Yellow);
        
        // 修改夜视仪属性
        if (_itemsDb.TryGetValue(ItemTpl.NIGHTVISION_L3HARRIS_GPNVG18_NIGHT_VISION_GOGGLES, out var nvgs))
        {
            logger.LogWithColor($"NVG 原始跳蚤可售: {nvgs.Properties.CanSellOnRagfair}", 
                LogTextColor.Red, LogBackgroundColor.Yellow);
            
            nvgs.Properties.CanSellOnRagfair = true;
        }
        
        return Task.CompletedTask;
    }
}

// 后期钩子：SPT 处理完成后执行
[Injectable(TypePriority = OnLoadOrder.PostSptModLoader + 1)]
public class LateHook(
    DatabaseServer databaseServer,
    ISptLogger<LateHook> logger
) : IOnLoad
{
    public Task OnLoad()
    {
        var items = databaseServer.GetTables().Templates.Items;
        
        // 验证之前的修改
        if (items.TryGetValue(ItemTpl.NIGHTVISION_L3HARRIS_GPNVG18_NIGHT_VISION_GOGGLES, out var nvgs))
        {
            logger.LogWithColor($"NVG 当前跳蚤可售: {nvgs.Properties.CanSellOnRagfair}", 
                LogTextColor.Green, LogBackgroundColor.Black);
        }
        
        return Task.CompletedTask;
    }
}
```

## 加载顺序示意图

```
服务器启动
    ↓
加载 JSON 数据库
    ↓
PostDBModLoader ← 你的早期钩子在这里执行
    ↓
SPT 核心逻辑处理
    ↓
PostSptModLoader ← 你的后期钩子在这里执行
    ↓
服务器就绪
```

## 小贴士

1. **选择正确时机**：根据需求选择合适的加载阶段
2. **使用 +1**：在系统默认顺序之后执行，确保数据已准备好
3. **彩色日志**：调试时使用 `LogWithColor` 让日志更醒目
4. **验证修改**：在后期钩子中验证早期修改是否生效
5. **避免重复操作**：同一数据不要在多个钩子中重复修改
