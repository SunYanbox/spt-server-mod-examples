# 数据库编辑

SPT 服务器启动时会把游戏数据（物品、商人、地图等）加载到内存中。你可以通过修改这些数据来改变游戏行为，比如修改物品属性、调整商人库存、改变地图配置等。

**源文件：** [2EditDatabase/EditDatabaseValues.cs](../2EditDatabase/EditDatabaseValues.cs)

## 数据库访问方式

### 方式一：DatabaseService（推荐）

高级 API，提供便捷的方法访问各类数据：

```csharp
using SPTarkov.Server.Core.Services;

public class MyMod(
    DatabaseService databaseService
) : IOnLoad
{
    public Task OnLoad()
    {
        // 获取全局配置
        var globals = databaseService.GetGlobals();
        
        // 获取藏身处数据
        var hideout = databaseService.GetHideout();
        
        // 获取机器人配置
        var bots = databaseService.GetBots();
        
        // 获取地图数据
        var locations = databaseService.GetLocations();
        
        return Task.CompletedTask;
    }
}
```

### 方式二：DatabaseServer（底层）

直接访问数据库表，适合操作物品模板：

```csharp
using SPTarkov.Server.Core.Servers;

public class MyMod(
    DatabaseServer databaseServer
) : IOnLoad
{
    public Task OnLoad()
    {
        // 获取所有物品模板
        var items = databaseServer.GetTables().Templates.Items;
        
        return Task.CompletedTask;
    }
}
```

## 常用数据编辑示例

### 1. 修改全局配置

全局配置包含很多游戏基础设置：

```csharp
var globals = databaseService.GetGlobals();

// 修改拾荒冷却时间为 1 秒
globals.Configuration.SavagePlayCooldown = 1;

// 修改跳蚤市场解锁等级为 1 级
globals.Configuration.RagFair.MinUserLevel = 1;

// 遍历修改跳蚤上架数量限制
foreach (var offerCountSettings in globals.Configuration.RagFair.MaxActiveOfferCount)
{
    offerCountSettings.Count = 20;  // 所有等级都能上架 20 个物品
}
```

### 2. 修改藏身处配置

让藏身处升级更快：

```csharp
using SPTarkov.Server.Core.Models.Enums.Hideout;

var hideout = databaseService.GetHideout();

// 找到厕所区域
var waterclosetArea = hideout.Areas.FirstOrDefault(area => area.Type == HideoutAreas.WaterCloset);

// 修改每个升级阶段
foreach (var stage in waterclosetArea.Stages)
{
    stage.Value.ConstructionTime = 60;  // 60 秒完成建造
    stage.Value.Requirements.Clear();    // 清空建造需求（免费升级！）
}
```

### 3. 修改机器人配置

让拾荒者更容易掉落好东西：

```csharp
using SPTarkov.Server.Core.Models.Enums;
using SPTarkov.Server.Core.Extensions;

var bots = databaseService.GetBots();
bots.Types.TryGetValue("assault", out var scav);  // assault = 拾荒者

// 让拾荒者更容易掉落旅游背包
scav.BotInventory.Equipment.TryGetValue(EquipmentSlots.Backpack, out var backpacks);
backpacks.AddOrUpdate(ItemTpl.BACKPACK_PILGRIM_TOURIST, 999999);  // 权重设得超高

// 让拾荒者总是携带 M4A1
scav.BotInventory.Equipment.TryGetValue(EquipmentSlots.FirstPrimaryWeapon, out var weapons);
weapons.AddOrUpdate(ItemTpl.ASSAULTRIFLE_COLT_M4A1_556X45_ASSAULT_RIFLE, 999999);

// 修改拾荒者名字
scav.FirstNames.Clear();
scav.FirstNames.Add("Gary");
```

### 4. 修改地图配置

调整海关地图的撤离点和 Boss 刷新：

```csharp
var locations = databaseService.GetLocations();
var customs = locations.Bigmap;  // 海关的内部名称是 "bigmap"

// 所有撤离点 100% 开放
foreach (var exit in customs.Base.Exits)
{
    exit.Chance = 100;
    exit.ChancePVE = 100;
}

// 让 Reshala（海关 Boss）100% 刷新
var bosses = customs.Base.BossLocationSpawn;
var reshala = bosses.FirstOrDefault(boss => boss.BossName == "bossBully");
reshala.BossChance = 100;

// 修改空投设置
var airdropSettings = customs.Base.AirdropParameters.First();
airdropSettings.PlaneAirdropChance = 1;  // 100% 概率（0-1 之间）
airdropSettings.PlaneAirdropStartMin = 1;  // 战局开始就能空投
```

### 5. 修改 BTR 配置

给 BTR 换个圣诞皮肤：

```csharp
var globals = databaseService.GetGlobals();
var btrSettings = globals.Configuration.BTRSettings;

// 获取森林地图的 BTR 配置
btrSettings.MapsConfigs.TryGetValue("Woods", out var woodsBtr);
woodsBtr.BtrSkin = "Tarcola";  // 圣诞皮肤
```

## 数据结构速查

| 数据类型 | 获取方法 | 主要内容 |
|---------|---------|---------|
| 全局配置 | `GetGlobals()` | 游戏基础设置、跳蚤、BTR 等 |
| 藏身处 | `GetHideout()` | 区域、工作站、配方 |
| 机器人 | `GetBots()` | 各类型机器人的装备和属性 |
| 地图 | `GetLocations()` | 各地图的配置、Boss、撤离点 |
| 物品模板 | `GetTables().Templates.Items` | 所有物品的属性定义 |
| 商人 | `GetTables().Traders` | 商人信息和库存 |

## 完整示例

```csharp
using SPTarkov.DI.Annotations;
using SPTarkov.Server.Core.DI;
using SPTarkov.Server.Core.Models.Spt.Mod;
using SPTarkov.Server.Core.Models.Utils;
using SPTarkov.Server.Core.Services;

namespace MyMod;

public record ModMetadata : AbstractModMetadata
{
    public override string ModGuid { get; init; } = "com.example.editdatabase";
    public override string Name { get; init; } = "数据库编辑示例";
    public override string Author { get; init; } = "开发者";
    public override SemanticVersioning.Version Version { get; init; } = new("1.0.0");
    public override SemanticVersioning.Range SptVersion { get; init; } = new("~4.0.0");
    public override string License { get; init; } = "MIT";
}

[Injectable(TypePriority = OnLoadOrder.PostDBModLoader + 1)]
public class EditDatabase(
    ISptLogger<EditDatabase> logger,
    DatabaseService databaseService
) : IOnLoad
{
    public Task OnLoad()
    {
        // 修改拾荒冷却
        var globals = databaseService.GetGlobals();
        globals.Configuration.SavagePlayCooldown = 1;
        
        // 修改海关撤离点
        var customs = databaseService.GetLocations().Bigmap;
        foreach (var exit in customs.Base.Exits)
        {
            exit.Chance = 100;
        }
        
        logger.Success("数据库修改完成！");
        return Task.CompletedTask;
    }
}
```

## 小贴士

1. **使用 TryGetValue**：访问字典时用 `TryGetValue` 更安全，避免空引用
2. **使用 ItemTpl 常量**：用 `ItemTpl.BACKPACK_PILGRIM_TOURIST` 比直接写字符串更清晰
3. **选择正确的加载时机**：数据库编辑通常用 `OnLoadOrder.PostDBModLoader + 1`
4. **做好日志记录**：修改完成后输出日志，方便调试
